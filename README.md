## About The Project

[![Product Name Screen Shot][product-screenshot]](image/logo.png)

This tool converts raw .xlsx sales data into:

A detailed processed report

A separate summary sheet with:

Total Revenue

Total Orders

Total Quantity Sold

Average Order Value

Best Customer
